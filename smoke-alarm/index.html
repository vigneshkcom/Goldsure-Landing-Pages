Before you buy a cheap DIY kit

Got hard-wired alarms? A DIY kit will not make you compliant.

DIY kit sellers are everywhere right now. For a lot of Queensland homes, that cheap kit is a false economy that leaves you non-compliant.

  • If your home already has hard-wired (240V) alarms, replacing them is electrical work
  • Queensland law requires that work to be done by a licensed electrician
  • A battery DIY kit there does not make the home compliant
  • Unlicensed electrical work can void your home insurance

We install to AS 3786 and hand you the certificate the same day, for $98 per alarm. Done once, done right.
