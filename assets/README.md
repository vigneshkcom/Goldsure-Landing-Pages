# Assets

This folder is for static media assets used by landing pages.

## Suggested structure

- `assets/branding/` → Goldsure website logo and branding assets
- `assets/hotwater/` → images/media used by `hotwater/index.html`
- `assets/smoke-alarm/` → images/media used by smoke alarm pages (optional)

You can upload and commit files here, then reference them in HTML with paths like:

- `/assets/branding/goldsure-logo.png`
- `/assets/branding/goldsure-wordmark.svg`
- `/assets/hotwater/logo.png`
- `/assets/hotwater/econova-290.webp`
- `/assets/hotwater/econova-330.webp`
